# querxnagios v1.0
Interact with Querx Smart Ethernet Sensors using Go (golang)

## compile
```
git clone https://github.com/egnite/querxnagios.git
cd querxnagios/check_querx
go install
cp $GOPATH/bin/check_querx /var/lib/nagios/plugins
```
